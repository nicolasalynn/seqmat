# SeqMat examples

| File | What it shows |
| --- | --- |
| [`kras_g12d_analysis.ipynb`](kras_g12d_analysis.ipynb) | End-to-end: chromosome coordinate → gene → mature mRNA → protein, comparing wild-type KRAS to the G12D oncogenic mutant. |
| [`seqmat_vs_biopython.ipynb`](seqmat_vs_biopython.ipynb) | Side-by-side: reverse complement, SNP throughput, mutation history, coordinate-preserving splicing, and a full coordinate→protein pipeline. With timings; honest about where each library wins. |
| [`basic_usage.py`](basic_usage.py) | `SeqMat` primitives: construction, mutations, complement, splicing. No data setup required. |
| [`data_inspection.py`](data_inspection.py) | Listing organisms, biotypes, gene counts via the discovery helpers. |

GitHub renders the notebook output inline — open it in the browser to see the results without running anything.
